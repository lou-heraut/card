# Catalogue des fiches CARD

Généré par `scripts/generate_catalog.py`, ne pas éditer à la main. Le décompte du corpus est dans le README.

Rangé par **domaine / phénomène** (le régime observé) ; la colonne *forme* distingue série, scalaire et courbe. Chaque fiche s'exécute via `card.extract(data, cards=[...])` ; *entrées* indique les colonnes que `data` doit contenir (cf. `rename=`). Détail d'une fiche : `card.info("nom")`.
